MCP is becoming the standard protocol for connecting AI agents (Claude, GPT, Copilot, Cursor) to external tools — and models are pulled into pipelines just as fast. The security tooling hasn't kept up:

- **150+ detection rules** across MCP servers and ML models
- A sweep of **112 artifacts** (33 MCP servers + 79 Hugging Face models) surfaced **3 critical** and **8 high** severity findings
- **12 published MCP CVEs** reproduced in our corpus
